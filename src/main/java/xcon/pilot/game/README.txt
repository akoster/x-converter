Game readme

A beginning network game, started off by StackOverflow user 'William'.

See:
http://stackoverflow.com/questions/1925801/can-someone-help-me-speed-up-my-netcode-in-java
http://slick.cokeandcode.com/
http://slick.javaunlimited.net/viewtopic.php?p=11655

Download the slick distribution and add all the lib/*.jar and /*.dll files to
your build path. Then modify the native library location of lwjgl.jar so it 
points to the directory where lwjgl.dll is located

To run, start a SlickServer and two SlickClients. Use 'a', 'w', 's', 'd' to move
around
